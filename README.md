# AI Repository  
AI 기반 ML/DL 학습 과정을 체계적으로 관리하기 위한 레포지토리이다.

## 브랜치 구조  
- **`main`**  
  opencv 학습 과정을 기록한다.  

- **`ML`**  
  머신러닝(ML) 학습 과정을 기록한다.  

- **`DL`**  
  딥러닝(DL) 학습 과정을 기록한다.  

- **`DL-Fastapi`**  
  서버, 모델, 프론트엔드를 통합한 웹 애플리케이션을 구현하여 모델을 웹페이지에서 실행할 수 있도록 한다.

- **`gen-vision`**
   비전(이미지) 기반 생성형 AI 기능을 개발한다.
